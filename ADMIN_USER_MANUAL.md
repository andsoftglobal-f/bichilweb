# Bichil Globus Админ Панел — Хэрэглэгчийн Гарын Авлага

Энэ баримт бичиг нь `bichilweb_admin` удирдлагын самбарыг өдөр тутам ашиглах хүмүүст зориулагдсан. Техникийн бус, практик заавар юм — код/архитектурын тайлбарыг [`SITE_ARCHITECTURE_HANDOVER.md`](SITE_ARCHITECTURE_HANDOVER.md)-с харна уу.

## 1. Нэвтрэх

1. Админ панелийн хаягийг (жишээ нь `https://admin.example.mn`) нээнэ.
2. Танд өгөгдсөн **нэвтрэх нэр** болон **нууц үг**-ээ оруулна.
3. Хэрэв энэ бол таны эхний нэвтрэлт бол системд автоматаар **"Нууц үг солих"** хуудас руу шилжинэ — шинэ, зөвхөн танд мэдэгдэх нууц үг тохируулна уу.
4. Нэвтэрсний дараа зүүн талд байрлах цэснээс бүх хэсэгт хандах боломжтой (таны эрхийн түвшингээс хамаарч зарим цэс харагдахгүй байж болно — доорх §4-г үзнэ үү).

**Гарах**: зүүн доод буланд байрлах **"Гарах"** товчийг дарна.

**Нууц үгээ мартсан бол**: Super Admin эрхтэй хэрэглэгчид Танд шинэ нууц үг тохируулж өгөх боломжтой (§4.3-г үзнэ үү) — өөрөө сэргээх боломж одоогоор байхгүй тул Super Admin-тайгаа холбогдоно уу.

## 2. Хянах самбар (Dashboard)

Нэвтэрсний дараа анх харагдах хуудас. Энд:
- Сүүлийн үеийн шинэчлэлтүүд (мэдээ, бүтээгдэхүүн, хуудас гэх мэт)
- Сайтын зочдын статистик (Analytics)
- Хурдан холбоосууд

харагдана.

## 3. Контент удирдлагын хэсгүүд

Зүүн цэсэн дэх бүлгүүд:

| Цэс | Юу удирддаг вэ |
|---|---|
| **Сайтын бүтэц** | Толгой хэсэг (header цэс), баннер слайдер, CTA слайдер, хөвөгч цэс, хөл хэсэг (footer), апп татах хэсэг, хамтрагчид, үзүүлэлтүүд, хуудас холбох |
| **Бүтээгдэхүүн** | Зээлийн бүтээгдэхүүнүүд, тэдгээрийн нөхцөл, баримт бичиг |
| **Үйлчилгээ** | Үйлчилгээний хуудсууд, карт, нөхцөл, баримт бичиг |
| **Байгууллага** | Бидний тухай хуудас, засаглал, удирдлагын бүрэлдэхүүн, салбарууд |
| **Мэдээ нийтлэл** | Мэдээ/нийтлэл нэмэх, засах, ангилал тохируулах |
| **Зар** | Зар сурталчилгааны удирдлага |
| **Хуудас удирдах** | Чөлөөт (custom) хуудас үүсгэх, засах |
| **Хүний нөөц** | HR тохиргоо, ажлын байрны зар, ирсэн CV/анкет харах |
| **Санхүү** | Валютын ханшийн тохиргоо |
| **Тохиргоо** | Лого, өнгөний тохиргоо, цэсний нэрс |
| **Зээлийн хүсэлт** | Сайтаас ирсэн зээлийн хүсэлтүүд |
| **Зээлийн тооцоолуур** | Тооцоолуурын тохиргоо |

Ихэнх хэсэгт **Шинэ нэмэх** товч, жагсаалт (хүснэгт), мөр тус бүрд **засах**/**устгах** үйлдэл байдаг — загвар бүх хэсэгт ижил.

## 4. Хэрэглэгч ба Эрхийн удирдлага (шинэ)

Энэ бол хамгийн сүүлд нэмэгдсэн, аюулгүй байдлын хамгийн чухал хэсэг тул тусад нь дэлгэрэнгүй тайлбарлав.

### 4.1 Хэн юу хийж чадах вэ

- **Super Admin** — бүх зүйлд хандах эрхтэй, шинэ хэрэглэгч үүсгэх/устгах/идэвхгүй болгох, role үүсгэх/засах цорын ганц эрхтэй хэрэглэгч.
- **Бусад role (жишээ нь "Admin")** — тухайн role-д ямар permission (жишээ нь "Мэдээ нэмэх", "Бүтээгдэхүүн засах") олгогдсоноос хамаарч харгалзах хэсгүүдэд л ажиллах боломжтой.

Зүүн цэсэн дэх **"Хэрэглэгчид"** болон **"Эрхийн түвшин"** цэс зөвхөн Super Admin-д харагдана.

### 4.2 Шинэ role (эрхийн түвшин) үүсгэх

1. **Эрхийн түвшин** цэс рүү орно.
2. **Шинэ role** товч дарна.
3. Нэр өгнө (жишээ нь "Контент менежер").
4. Модуль тус бүрээр ямар үйлдэл (Нэмэх/Засах/Устгах/Харах) зөвшөөрөхөө checkbox-оор тэмдэглэнэ.
5. **Хадгалах**.

> Систем анх суулгагдахад **"Admin"** нэртэй role автоматаар үүсч, бүх модуль дээр бүрэн эрхтэй байдаг. Хэрэв танд илүү нарийвчилсан, хязгаарлагдмал эрхтэй role хэрэгтэй бол шинээр үүсгэж, тухайн хэрэглэгчид оноогоорой.

### 4.3 Шинэ хэрэглэгч үүсгэх

1. **Хэрэглэгчид** цэс рүү орно.
2. **Шинэ хэрэглэгч** товч дарна.
3. Нэвтрэх нэр, и-мэйл, нэр/овог, анхны нууц үг оруулна, харгалзах **role**-ыг сонгоно.
4. **Хадгалах** — хэрэглэгч эхний удаа нэвтрэхдээ нууц үгээ солих шаардлагатай болно (автоматаар).

### 4.4 Хэрэглэгчийг идэвхгүй болгох

Устгахын оронд **идэвхгүй болгохыг** зөвлөж байна (устгах боломж санаатайгаар байхгүй — түүхэн мэдээллийг хадгалах үүднээс):

1. **Хэрэглэгчид** цэсэн дэх жагсаалтаас тухайн хэрэглэгчийг **засах**.
2. **"Идэвхтэй"** checkbox-ыг арилгана.
3. **Хадгалах**.

Идэвхгүй болсон хэрэглэгч цаашид нэвтрэх боломжгүй болно. (Та өөрийн эрхээ идэвхгүй болгох боломжгүй — систем үүнийг зөвшөөрөхгүй.)

### 4.5 Хэрэглэгчийн нууц үг дахин тохируулах

Хэрэглэгч нууц үгээ мартсан бол Super Admin дараах байдлаар шинэ нууц үг өгч болно:

1. **Хэрэглэгчид** цэсэн дэх жагсаалтаас тухайн хэрэглэгчийн мөр дээрх **харах (нүд)** дүрс товчийг дарна.
2. Шинэ түр нууц үг оруулна.
3. **Тохируулах**.

Хэрэглэгч дараагийн нэвтрэлтдээ энэ нууц үгээ солих шаардлагатай болно.

## 5. Түгээмэл асуулт

**"Нэвтэрсэн ч зарим цэс дээр эрхгүй гэсэн алдаа гарч байна"** — таны role-д тухайн модулийн permission олгогдоогүй байна. Super Admin-тайгаа холбогдож Role Management-с эрх нэмүүлнэ үү.

**"Хадгалах товч дарахад юу ч болохгүй байна"** — интернэт холболтоо шалгаад дахин оролдоно уу; асуудал үргэлжилбэл техникийн багтай холбогдоно уу.

**"Сессийн хугацаа дуусаад автоматаар гарлаа"** — аюулгүй байдлын үүднээс нэвтрэлт тодорхой хугацааны дараа автоматаар шинэчлэгддэг (хэрэглэгч анзаарахгүйгээр) боловч удаан идэвхгүй байвал дахин нэвтрэх шаардлагатай болно. Энэ нь хэвийн үзэгдэл.
